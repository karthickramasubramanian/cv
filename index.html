Jeeves’ EazyStock
Jeeves’ EazyStock is a customized application on Jeeves Universal ERP system that offers integration with EazyStock (ES) inventory optimization software.
Jeeves' EazyStock adapter is intended for exporting the relevant data from Jeeves and for importing
inventory optimization data from ES according to standardized file formats defined for ES. MS SSIS was used as the integration tool.

Jeeves’ EasyInvoice
Jeeves’ EasyInvoice is a customized application on Jeeves Universal ERP system that offers automation of customer invoices, without any manual intervention.
Invoices are created and the invoice reports are generated using a schedule. The generated reports can be moved to a FTP folder, uploaded as a PDM document in Jeeves and sent via email.
All the above actions are automated based on the users’ configuration in Jeeves system. The reports are designed using SSRS.
